# Personal Income Tax Calculator(python)
This program is a python progarmme that calculates U.S. personal income tax (2009) based on filing status and taxable income.
## Features
- Calculates personal income tax based on filing status and taxable income
- Uses conditional statement to apply the correct tax brackets
- Includes basic error handling to prevent invalid input
- Demonstrates the use of python control flow and user input

## Filing Status Codes
0 = single
1 = Married filing jointly or qualifying widow(er)
2 =  Married filing separately
3 = Head of household
## How to Run
Run the program and follow the prompts to enter your filing status and taxable income.
